## 2024-10-23
main.jsからコードを取り込みたい時は `src`属性を使う

```
<body>
  <script src="main.js">
</body>
```

jsファイルの一行目は`'use strict';`と書くこと。  
これは良くない書き方をしているときにエラーを表示してくれる厳格モードになってくれる。
